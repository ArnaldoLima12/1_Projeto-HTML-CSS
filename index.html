HTML 5

O HTML 5 é a mais recente versão da linguagem de marcação de hipertexto (HTML), que é usada para estruturar e exibir conteúdo na Web. Ele foi lançado em 2014 e introduziu muitos recursos novos e aprimorados em comparação com as versões anteriores do HTML.

CSS

O CSS permite que você controle o estilo de vários elementos em uma página da web, como a cor do texto, a fonte, a posição de um elemento na página, entre outros. Isso significa que você pode definir as propriedades de estilo para um elemento em um único lugar e aplicá-las a todos os lugares onde esse elemento é usado na página.

JavaScript

JavaScript é uma linguagem de programação de alto nível, dinâmica e interpretada que é usada principalmente para adicionar interatividade a páginas da web. Ele é executado no lado do cliente, ou seja, no navegador do usuário, e é amplamente utilizado para criar efeitos visuais, animações, validar formulários, interagir com APIs e muito mais.
